# alexander urbach :: 17099450a :: concert photography portfolio

concert photography portfolio featuring live shows from ottawa, montreal, and toronto.

## about

a personal project built from scratch using vanilla HTML, CSS, and JavaScript. I will be updating the website and adding photos over time!

## features

- responsive, mobile-first layout  
- interactive galleries with drag + momentum  
- fullscreen lightbox viewer  
- collection-based navigation  
- contact form for bookings  
- optimized images + metadata  

## tech stack

- HTML5  
- CSS3 (flexbox, grid, custom properties)  
- vanilla JavaScript (ES6+)
